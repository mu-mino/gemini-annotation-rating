# Annotation Tool

## Overview

This repository hosts a public annotation and rating tool designed for the **human evaluation of outputs generated by the `blocks-to-xml-pipeline` branch** in the [Auto-Classify-Hadiths](https://github.com/muhammedemineser/Auto-Classify-Hadiths) repository.

It provides a systematic workflow for manually reviewing and scoring the data processing results from the main project.

---

## Role in the Project Ecosystem

The `annotation-tool` serves as a **complementary review utility** for the main Hadith classification project:

* **Core Pipeline (AI Classification):** Managed in [Auto-Classify-Hadiths](https://github.com/mu-mino/Auto-Classify-Hadiths).
* **Validation Layer (This Repository):** Enables manual verification of the pipeline's outputs.
